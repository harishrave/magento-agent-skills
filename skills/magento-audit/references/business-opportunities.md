# Business Opportunities

**Category 11** — strategic improvements beyond defect fixes.

These are **opportunities**, not defects. Use severity **Info** or **Low** unless tied to measurable revenue risk.

## Suggest only when supported

Base suggestions on evidence from other audit categories — e.g. slow checkout (performance) → checkout optimization opportunity.

**Do not invent** revenue figures or conversion rates.

## Opportunity areas

| Area | Example opportunity | Trigger from audit |
|---|---|---|
| Checkout optimization | One-page checkout, address autocomplete | High checkout abandonment signals, many steps |
| Search | Live Search, typo tolerance | MySQL search, poor PLP UX |
| Layered navigation | Ajax filters, SEO-friendly URLs | Performance + SEO findings |
| B2B | Company accounts, quotes, approval | B2B business model stated by user |
| Adobe Commerce features | Staging, rewards, customer segments | On Open Source with enterprise needs |
| Customer experience | Guest checkout, returns portal | UX findings |
| Marketing | Cart rules complexity, personalization | Extension sprawl |
| Performance | Hyvä migration, CDN, image optimization | CWV failures |
| SEO | Structured data, faceted URL strategy | SEO audit gaps |
| Conversion | Trust badges, stock visibility, urgency | UX review |
| Admin productivity | Mass actions, better grids | Admin UX findings |

## Report format

| Opportunity | Evidence | Expected benefit | Effort | Priority |
|---|---|---|---|---|
| Enable Live Search | MySQL search engine configured | Faster search, better relevance | M | Medium |

## Roadmap placement

Place in **Recommended roadmap** section of [audit-report-template.md](audit-report-template.md) — not mixed with Critical security findings.
